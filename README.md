# NBRadioGroup
A Simple RadioGroup component for NativeBase


### Example Usage

```jsx harmony
<RadioGroup

    //Direction
    row

    //Callback
    onChangeValue={(activeGender) => this.setState({activeGender})}

    //Default
    selectedValue="male">

    <RadioButton label="Male" value="male"/>
    <RadioButton label="Female" value="female"/>

</RadioGroup>
```
